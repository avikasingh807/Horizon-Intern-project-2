# 🌿 Skincare E-commerce Product Page

A responsive and modern skincare product page built using HTML, CSS, and JavaScript.  
This project focuses on clean UI, user experience, and real-world e-commerce features.

---

## ✨ Features

- 🖼️ Product Image Gallery (click to change image)
- 🛒 Add to Cart (with Local Storage)
- ❤️ Wishlist Button
- ⭐ Customer Reviews Section
- 📱 Fully Responsive Design (Mobile + Desktop)
- 🎨 Clean & Aesthetic UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM + Local Storage)
